# World Clock Widget 需求文档

## 1. 组件概述

世界时钟组件，用于同时显示多个城市的当前时间。

## 2. 布局尺寸

- **固定尺寸**: 1×2 (SMALL_WIDGET)
- 宽度: 1 Grid Unit
- 高度: 2 Grid Units

## 3. 功能需求

### 3.1 默认城市
组件默认显示以下三个城市：
1. 北京 (Asia/Shanghai)
2. 纽约 (America/New_York)
3. 伦敦 (Europe/London)

### 3.2 显示内容
每个城市显示：
- 城市名称（中文）
- 当前时间（HH:MM 格式，24小时制）
- 与本地的时差（如 +8h、-5h）
- 昼夜状态指示（白天/夜晚）

### 3.3 右键菜单
- **编辑**: 打开编辑对话框，可选择/更换显示的城市
- **删除**: 删除组件（继承自默认菜单）

## 4. 样式规范

### 4.1 整体风格
- 使用项目统一的 CSS 变量
- 毛玻璃背景效果
- 三列布局，每列显示一个城市

### 4.2 昼夜指示
- 白天: 黄色太阳图标 + 黄色光晕点
- 夜晚: 紫色月亮图标 + 紫色光晕点

### 4.3 动画
- 时间每秒更新
- 光晕点脉动效果

## 5. 数据存储

- 城市配置存储在 `chrome.storage.local`
- Key: `world-clock-cities`
- 格式: `Array<{ name: string; timezone: string; label: string }>`

## 6. 可选城市列表

支持的主要时区：
- 亚洲: 北京、东京、首尔、新加坡、香港、迪拜
- 欧洲: 伦敦、巴黎、柏林、莫斯科
- 美洲: 纽约、洛杉矶、芝加哥、多伦多、圣保罗
- 大洋洲: 悉尼、墨尔本
